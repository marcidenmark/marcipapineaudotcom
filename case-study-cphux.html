CPHUX
Denmarks largest UX Community
Business Case

CPHUX is a Copenhagen based startup working to raise the industry standard of UX design in Copenhagen by uniting designers and businesses.

The Challenge

CPHUX was in its startup phase. They needed a website and an application where people could sign up and pay. And the quicker the beter. Payments would be used to develop the product.

Project Scope & Objectives

There were a variety of things that would be needed to bring in paying members.

» A fully responsive website– optimized for both desktop and mobile devices.

» A signup flow to guide the user in joining, paying, and registering as a new user.

» The user would also need to be able to sign in and out of application.

» An HTML template needed to be created in order to create new pages on the website when needed.

» A style guide needed to be created to keep all added features consistant.

The Outcome

✓ Working MVP where a user could signup and pay.

✓ When I left there was a working style guide written in code to carry on with the design.

✓ Modular design & code for easy adoption to new pages and components.

✓ Product optimised to work on mobile and desktop.
