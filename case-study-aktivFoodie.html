AktivFoodie
Restaurant review app for athletic travellers
Business Case

AktivFoodie is an international company based in Copenhagen. It provides a platform for unprofessional athletes to find places to eat while traveling for sports events, such as marathons or Ironmans…

AktivFoodie helps traveling athletes to find healthy food through user reviews. The focus is finding highly rated restaurant reviews with an athlete’s need in focus.

The Challenge

AktivFoodie had the start of a Wordpress app that needed some help. They needed the home screen to have more information, and it needed to be designed. It needed CSS customized, and more added to the page so in was inviting. The app also needed the ability for users to sign in and upload reviews including adding an image.

Project Scope & Objectives

The app needed to have an inviting home page. It should guide users to read reviews and add create their own if needed. It was a MUST to work on mobile, as that is where the majority of users are. But not only mobile, as users are also active on the platform from hotel rooms, on their computers.

The Outcome

I was able to get an inviting and easy to use app by the end of a 2 week period. It included:

✓ A product that worked great on mobile and desktop

✓ A home screen showing testimonials and reviews

✓ A user can signup

✓ A user can write reviews and upload reviews

✓ A user can find directions directly with google map integration
